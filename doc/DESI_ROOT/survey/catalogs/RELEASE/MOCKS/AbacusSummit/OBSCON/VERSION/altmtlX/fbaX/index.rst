

====
fbaX
====

``${DESI_ROOT}/survey/catalogs/RELEASE/MOCKS/AbacusSummit/OBSCON/VERSION/altmtlX/fbaX`` contains the fba compiled data of potential and assigned data, matched to the parent forFA file.

Two different files can be found here, or all potential assigments matched to the data (with duplicates) or only those that were assigned to a fiber (with duplicates)
