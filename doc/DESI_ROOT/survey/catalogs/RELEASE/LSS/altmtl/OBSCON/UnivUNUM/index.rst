========
UnivUNUM
========

``${DESI_ROOT}/survey/catalogs/RELEASE/LSS/altmtl/OBSCON/UnivUNUM`` refers to 128 Univ ``{UNUM}`` directories for a given ``{OBSCON}`` program. Each of these contains the alternative MTL history for the particular seeding of subpriorities. in fa directory, we include all the individual fiber assigment executions ordered by date. In main, we include the merged target list of the observations with the same format as the observed data.
