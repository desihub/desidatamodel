=======
PHOTSYS
=======

``$DESI_ROOT/vac/RELEASE/lss/SPECPROD/SURVEY/VERSION/altmtl/PHOTSYS/UnivUNUM/main/PHOTSYS`` contains the Merged Target List (MTL) ledgers ( :ref:`see, e.g., <mtl_ledger>`) with subpriorities re-seeded, but following the fiber-specific observational information on whether the target assigned to a particular positioner was successful. There are 128 realizations, each in a separate ``Univ{UNUM}`` directory and with ``{PHOTSYS}`` being bright or dark. 

The filename for each ledger resembles mtl-PHOTSYS-hp-HPX.ecsv is the healpixel number and PHOTSYS dark or bright.
