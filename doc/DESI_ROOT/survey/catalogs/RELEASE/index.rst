=======
RELEASE
=======

``${DESI_ROOT}/survey/catalogs/RELEASE`` contains the LSS catalogs for bright (BGS) and dark (LRG, ELG, QSO) used for clustering analysis. The value of ``RELEASE`` will be ``dr1`` or ``dr3``, and so on. In addition, QSO directory contains all the observations that pass the automated QSO selection criteria and mocks directory includes the various galaxy mock catalogs employed for analysis validation and covariance estimation.
