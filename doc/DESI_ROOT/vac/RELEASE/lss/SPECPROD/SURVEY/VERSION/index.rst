=======
VERSION
=======

``$DESI_ROOT/vac/RELEASE/lss/SPECPROD/SURVEY/VERSION`` contains directories for the end-products LSS catalogs (``LSScats``), inputs based on processing of the spectroscopic reductions (``inputs_wspec``) and from processing of fiberassign information (``potential_assignments``), as well as the Alternative Merged Target List (MTL) ledgers (``altmtl``).
