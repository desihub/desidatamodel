======
SURVEY
======

``$DESI_ROOT/vac/RELEASE/lss/SPECPROD/SURVEY`` contains different possible versions of the LSS catalogs, given a production run and a tile selection. For EDR, there is one single ``VERSION`` named ``v2`` (we use v2 in order to distinguish these catalogs from an internal DESI earlier v1 version).
