======
SURVEY
======

``$DESI_ROOT/vac/RELEASE/lss/SPECPROD/SURVEY`` contains different possible versions of the LSS catalogs, given a production run and a tile selection. For EDR, there is one single ``VERSION`` named :envvar:`v1`, but in future releases more version could be published.
