========
UnivUNUM
========

``$DESI_ROOT/vac/RELEASE/lss/SPECPROD/SURVEY/VERSION/altmtl/UnivUNUM`` refers to 128 Univ ``{UNUM}`` directories. Each contains a ``dark`` and a ``bright`` directory. Each of these contains the alternative MTL history for the particular seeding of subpriorities.
