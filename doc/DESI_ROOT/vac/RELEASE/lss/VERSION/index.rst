=======
VERSION
=======

``$DESI_ROOT/vac/RELEASE/lss/VERSION`` contains directories for the end-products LSS catalogs (``LSScats``), inputs based on processing of the spectroscopic reductions (``inputs_wspec``, only in EDR) and from processing of fiberassign information (``potential_assignments``, only in EDR), as well as the Alternative Merged Target List (MTL) ledgers (``altmtl``, only in EDR).
