=======
version
=======

:envvar:`version` contains directories for the end-products LSS catalogs (``LSScats``), inputs based on processing of the spectroscopic reductions (``inputs_wspec``) and from processing of fiberassign information (``potential_assignments``), as well as the Alternative Merged Target List (MTL) ledgers (``altmtl``). Products come from the root :envvar:`DESI_ROOT`.
