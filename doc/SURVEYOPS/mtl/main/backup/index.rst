======
backup
======

The ``backup`` directory hosts MTL ledgers for the DESI backup program.
Targets are stored in ledgers that are split by HEALPixel (in the NESTED
scheme) at a resolution of nside=32.

The filename for each ledger resembles mtl-backup-hp-HPX.ecsv, where HPX
is the healpixel number.
