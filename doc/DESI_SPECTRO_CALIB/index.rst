==================
DESI_SPECTRO_CALIB
==================

:envvar:`DESI_SPECTRO_CALIB` contains spectrograph calibration data.
This directory tree is only partially documented here.

:envvar:`DESI_SPECTRO_CALIB` will contain both a root path and a version
string, for example ``${DESI_ROOT}/spectro/desi_spectro_calib/0.4.0`` in
DR1.
