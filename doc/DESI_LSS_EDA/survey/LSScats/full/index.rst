====
full
====

:envvar:`full` contains information on all targets identified as reachable by DESI fiberassign, split by target types (LRG, ELG...), including many columns of photometrically and spectroscopically derived information. These files are available before (``_noveto`` is in the file name) and after vetos related to angular coordinates are applied. It includes both data for randoms and data
