===========
DESI_TARGET
===========

:envvar:`DESI_TARGET` contains target selection and fiber assignment data,
with the canonical location of ``$DESI_ROOT/TS/target``. Here, ``TS`` is, e.g.,
`da/ets` for DESI early target selection.
