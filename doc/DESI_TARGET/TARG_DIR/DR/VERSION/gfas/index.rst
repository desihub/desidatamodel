====
gfas
====

The ``gfas`` directory contains targets used to guide, focus and align
the DESI instrument. GFA targets are stored in files that are grouped by
(nested) HEALPixel number in filenames that
resemble gfas-hp-{``HP``}.fits, where ``HP`` is the HEALPixel number.
